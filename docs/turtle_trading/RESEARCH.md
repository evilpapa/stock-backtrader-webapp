# 海龟交易法则策略

“海龟法则”是一个完整、系统且机械化的趋势跟踪交易策略，其核心思想在于捕获市场中的重大趋势行情，并通过严格的资金管理和风险控制来长期获利。

## 一、传奇的起源：一场赌局

1983年，美国期货界的两位顶尖交易员理查德·丹尼斯和威廉·埃克哈特之间发生了一场著名的辩论。

辩论的焦点是：

> 伟大的交易员到底是天生的，还是可以后天培养的？

丹尼斯相信，交易能力可以通过教授一套规则系统来培养。他打赌说，就像在新加坡农场养殖海龟一样，他也可以“养殖”出优秀的交易员。

埃克哈特则更倾向于认为，交易需要某种天生的直觉和才能。

为了验证这一观点，丹尼斯在报纸上刊登广告，招募了一批完全没有交易经验的普通人（包括演员、律师、会计等）。他们接受了丹尼斯两周的培训，学习了一套完整的交易系统法则。之后，丹尼斯给他们真实的资金账户进行交易。

结果，这批被称为“海龟”的交易员在随后的几年中取得了惊人的平均年化复利回报率80%，总额盈利超过1亿美元，成功证明了用一套可量化的系统是可以培养出优秀交易员的。这场实验也成为了交易史上最著名的实验之一。

## 二、海龟交易法则的核心规则

海龟交易法则是一个完整的体系，主要包括以下几个部分：

### 1. 市场选择与流动性

交易哪些市场？——高流动性的市场。最初主要在期货市场，如原油、外汇、黄金、咖啡、国债等。这套理念同样适用于股票、ETF等流动性好的市场。

### 2. 头寸规模确定

基于波动性的风险均衡：海龟们不按“买多少手”来交易，而是按“愿意承担多少风险”来交易。

首先，计算市场的真实波动幅度（N值）：

N = (前20个交易日的) 平均真实波动范围（ATR）

ATR反映了市场每日价格波动的平均幅度，是衡量波动性的关键指标。

然后，根据账户规模和风险计算头寸规模：

头寸规模单位 = (账户资金的1% ) / (N* 每点价值)

这意味着海龟们将任何单笔交易的最大风险控制在账户总资金的1%。如果市场波动性（N值）变大，他们就会自动减少交易的手数，以保证潜在的亏损额不变。这实现了在不同波动性的市场中风险暴露的一致性。

### 3. 入市信号（何时买卖？）

海龟系统有两套入市系统，交易员可以任选其一，但必须严格遵守。

系统一：以20日突破为基础（短期系统）

- 做多：当价格突破过去20个交易日的最高价时。
- 做空：当价格跌破过去20个交易日的最低价时。

只要价格创出20日新高/新低，就入场，即使是跳空缺口也照跟不误。

系统二：以55日突破为基础（长期系统）

规则同上，但参考的是55日的最高价和最低价。

此系统发出的信号更少，但捕捉的趋势更大，交易成本也更低。

### 4. 加仓策略（如何扩大战果？）

金字塔加仓：趋势确认后，逐步加仓。

每次加仓的间隔是 0.5个N值（即价格向有利方向移动了0.5倍ATR）。

最多允许加仓3次，加上初始仓位，总共4个单位仓位。这样既抓住了趋势，又避免了在不利位置过度重仓。

### 5. 止损策略（如何控制损失？）

任何一笔交易的最大风险绝不超出账户的2%。

对于每一个持仓单位，止损点设置在 2个N值 的位置。

例如：买入价为100，N值为2，则止损价为 100-2*2 = 96。

如果加了仓，所有仓位的止损点会统一上移至最新仓位入场价下方的2N处，实现移动止损，保护浮动盈利。

### 6. 离市策略（何时退出盈利？）

这是海龟法则最难执行的部分：逆势出场。

- 多头头寸：当价格跌破过去10日（系统一）或20日（系统二）的最低价时离场。
- 空头头寸：当价格升破过去10日（系统一）或20日（系统二）的最高价时离场。

> 注意：离市信号与入市信号无关，它只关心趋势是否已经反转。为了抓住整个趋势的大部分，你必须愿意放弃相当一部分已得的利润。


## 三、海龟交易法的优缺点

优点：

1. 完全系统化：杜绝了情绪干扰，一切操作有据可依。
2. 风险控制极致：1%的风险规则和2N止损确保了生存是第一要务。
3. 抓住大趋势：在出现单边大行情时，收益会非常惊人。
4. 普适性：原理适用于任何有趋势的市场和时间框架。

缺点：

1. 震荡市是克星：在缺乏趋势的震荡行情中，会连续出现多次“假突破”，导致频繁的小额亏损（称为“回撤期”）。这对交易者的心理是极大的考验。
2. 需要极强的纪律性：看着利润回吐仍能坚持执行规则，需要反人性的毅力。
3. 高资金要求：由于需要分散投资多个不相关的市场来平滑曲线，并且要承受较大的回撤，对初始资金量要求较高。

## 总结

海龟交易法则不仅仅是一套交易策略，更是一种交易哲学。它强调了纪律、系统化、风险管理和一致性的极端重要性。即使你不直接使用它的具体规则，其核心思想——“截断亏损，让利润奔跑”——也是所有趋势跟踪者乃至广大交易者的金科玉律。

它告诉我们，成功的交易并非依赖于预测市场的魔法，而是依赖于对概率的把握和严格执行计划的能力。

# xtquant.xtdata 财务数据接口

来源：[XtQuant.XtData 行情模块](https://dict.thinktrader.net/nativeApi/xtdata.html?id=olYCD5)

## 函数速查

- `get_financial_data(stock_list, table_list=[], start_time='', end_time='', report_type='report_time')`
- `download_financial_data(stock_list, table_list=[])`
- `download_financial_data2(stock_list, table_list=[], start_time='', end_time='', callback=None)`

## get_financial_data

- 作用：读取财务数据。
- 参数：
  - `stock_list` 合约代码列表
  - `table_list` 财务表名列表
  - `start_time`
  - `end_time`
  - `report_type`
- `report_type` 常见值：
  - `report_time` 按截止日期筛选
  - `announce_time` 按披露日期筛选

### 支持的财务表

- `Balance` 资产负债表
- `Income` 利润表
- `CashFlow` 现金流量表
- `Capital` 股本表
- `Holdernum` 股东数
- `Top10holder` 十大股东
- `Top10flowholder` 十大流通股东
- `Pershareindex` 每股指标

### 返回结构

- 顶层：`dict[stock] -> datas`
- 第二层：`datas` 为 `dict[table] -> DataFrame`
- 每个 `DataFrame` 的列定义参考附录中的财务字段列表。

## download_financial_data

- 作用：下载财务数据。
- 参数：
  - `stock_list`
  - `table_list`
- 备注：同步执行，下载完成后返回。

## download_financial_data2

- 作用：按时间范围下载财务数据。
- 参数：
  - `stock_list`
  - `table_list`
  - `start_time`
  - `end_time`
  - `callback`
- 时间筛选依据：文档说明按 `m_anntime` 披露日期筛选。
- 进度回调字段：
  - `total`
  - `finished`
  - `stockcode`
  - `message`

## 使用建议

- 用户问“取不到财务数据”时，优先检查是否先执行过 `download_financial_data*`。
- 用户问“按公告日还是报告期筛选”时，优先解释 `report_type`。
- 用户问具体财务字段时，直接跳转到 `5-附录-财务字段与合约字段.md`。

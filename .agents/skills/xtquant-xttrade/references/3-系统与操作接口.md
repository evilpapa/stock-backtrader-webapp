# xtquant.xttrader 系统与操作接口

来源：[XtQuant.Xttrade 交易模块](https://dict.thinktrader.net/nativeApi/xttrader.html?id=olYCD5)

## 系统设置接口

### XtQuantTrader(path, session_id)

- 作用：创建 API 实例。
- `path`：`userdata_mini` 完整路径。
- `session_id`：与 MiniQmt 通信的会话 ID。
- 备注：通常只需创建一个 trader 实例。

### register_callback(callback)

- 作用：注册 `XtQuantTraderCallback` 回调类实例。

### start()

- 作用：启动交易线程，准备交易环境。

### connect()

- 作用：连接 `MiniQmt`。
- 返回：成功 `0`，失败非 `0`。
- 备注：断开后不会自动重连。

### stop()

- 作用：停止 API。

### run_forever()

- 作用：阻塞当前线程，接收主推，直到 `stop()`。

### set_relaxed_response_order_enabled(enabled)

- 作用：开启主动请求接口专用线程，放宽返回时序。
- 适用：在推送回调中调用同步查询可能卡住时。
- 代价：查询返回与推送顺序可能不严格一致。

## 操作接口

### subscribe(account)

- 作用：订阅账号信息，包括资金、委托、成交、持仓主推。
- 返回：成功 `0`，失败 `-1`。

### unsubscribe(account)

- 作用：反订阅账号信息。
- 返回：成功 `0`，失败 `-1`。

### order_stock(account, stock_code, order_type, order_volume, price_type, price, strategy_name, order_remark)

- 作用：同步报单。
- 返回：大于 `0` 的订单编号表示成功，`-1` 表示失败。

### order_stock_async(account, stock_code, order_type, order_volume, price_type, price, strategy_name, order_remark)

- 作用：异步报单。
- 返回：大于 `0` 的请求序号 `seq` 表示成功，`-1` 表示失败。
- 后续：正常情况下会收到 `on_order_stock_async_response` 推送。

### cancel_order_stock(account, order_id)

- 作用：按订单编号同步撤单。
- 返回：`0` 成功，`-1` 失败。

### cancel_order_stock_sysid(account, market, order_sysid)

- 作用：按柜台合同编号同步撤单。
- 返回：`0` 成功，`-1` 失败。

### cancel_order_stock_async(account, order_id)

- 作用：按订单编号异步撤单。
- 返回：大于 `0` 的撤单请求序号，`-1` 失败。

### cancel_order_stock_sysid_async(account, market, order_sysid)

- 作用：按柜台合同编号异步撤单。
- 返回：大于 `0` 的撤单请求序号，`-1` 失败。

### fund_transfer(account, transfer_direction, price)

- 作用：资金划拨。
- 返回：`(success, msg)`

### sync_transaction_from_external(operation, data_type, account, deal_list)

- 作用：外部交易数据录入。
- `operation`：如 `UPDATE`、`REPLACE`、`ADD`、`DELETE`
- `data_type`：文档示例为 `DEAL`
- 返回：结果字典。

## 最小流程示意

```python
from xtquant.xttrader import XtQuantTrader, XtQuantTraderCallback
from xtquant.xttype import StockAccount

path = r'D:\迅投极速交易终端 睿智融科版\userdata_mini'
session_id = 123456
xt_trader = XtQuantTrader(path, session_id)
xt_trader.register_callback(XtQuantTraderCallback())
xt_trader.start()
xt_trader.connect()

account = StockAccount('1000000365')
xt_trader.subscribe(account)
```

## 使用建议

- 用户说“下单没反应”时，先检查是否连接和订阅成功。
- 用户说“撤单失败”时，先确认用的是 `order_id` 还是 `order_sysid`。
- 用户说“异步没有反馈”时，先检查是否实现了对应 callback。

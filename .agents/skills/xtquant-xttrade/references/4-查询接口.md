# xtquant.xttrader 查询接口

来源：[XtQuant.Xttrade 交易模块](https://dict.thinktrader.net/nativeApi/xttrader.html?id=olYCD5)

## 股票与期货查询接口

### query_stock_asset(account)

- 作用：查询资产。
- 返回：`XtAsset` 或 `None`

### query_stock_orders(account, cancelable_only=False)

- 作用：查询当日所有委托。
- `cancelable_only=True` 时仅查可撤委托。
- 返回：`list[XtOrder]` 或 `None`

### query_stock_trades(account)

- 作用：查询当日所有成交。
- 返回：`list[XtTrade]` 或 `None`

### query_stock_positions(account)

- 作用：查询最新持仓。
- 返回：`list[XtPosition]` 或 `None`

### query_position_statistics(account)

- 作用：查询期货账号持仓统计。
- 返回：`list[XtPositionStatistics]` 或 `None`
- 常见账户：`StockAccount('账号', 'FUTURE')`

## 信用查询接口

### query_credit_detail(account)

- 作用：查询信用资产。
- 返回：`XtCreditDetail` 或 `None`
- 常见账户：`StockAccount('账号', 'CREDIT')`

### query_stk_compacts(account)

- 作用：查询负债合约。
- 返回：`list[StkCompacts]` 或 `None`

### query_credit_subjects(account)

- 作用：查询融资融券标的。
- 返回：`list[CreditSubjects]` 或 `None`

### query_credit_slo_code(account)

- 作用：查询可融券数据。
- 返回：`list[CreditSloCode]` 或 `None`

### query_credit_assure(account)

- 作用：查询标的担保品。
- 返回：`list[CreditAssure]` 或 `None`

## 其他查询接口

### query_new_purchase_limit(account)

- 作用：查询新股申购额度。
- 返回：`dict`
- 常见键：
  - `KCB`
  - `SH`
  - `SZ`

### query_ipo_data()

- 作用：查询当日新股新债信息。
- 返回：`dict[stock_code] -> info`
- 常见字段：
  - `name`
  - `type`
  - `minPurchaseNum`
  - `maxPurchaseNum`
  - `purchaseDate`
  - `issuePrice`

### query_account_infos()

- 作用：查询所有资金账号。
- 返回：`list[XtAccountInfo]`

### query_account_status()

- 作用：查询所有账号状态。
- 返回：`list[XtAccountStatus]`

### query_com_fund(account)

- 作用：查询普通柜台资金。
- 返回：结果字典。
- 常见字段：
  - `success`
  - `error`
  - `currentBalance`
  - `enableBalance`
  - `fetchBalance`
  - `assetBalance`
  - `marketValue`
  - `debt`

### query_com_position(account)

- 作用：查询普通柜台持仓。
- 返回：持仓字典列表。
- 常见字段：
  - `success`
  - `error`
  - `stockAccount`
  - `exchangeType`
  - `stockCode`
  - `stockName`
  - `totalAmt`
  - `enableAmount`
  - `lastPrice`
  - `costPrice`
  - `income`
  - `incomeRate`
  - `marketValue`
  - `costBalance`

### export_data(account, result_path, data_type, start_time=None, end_time=None, user_param={})

- 作用：导出通用数据到文件。
- 返回：结果字典。

### query_data(account, result_path, data_type, start_time=None, end_time=None, user_param={})

- 作用：导出后读取数据内容，再删除导出文件。
- 返回：数据信息字典或表结构内容。

## 使用建议

- 查询失败或为空时，文档中很多接口都用 `None` 表示失败或空结果，回答时要明确这两种可能。
- 用户问信用查询时，先确认账户类型是否为 `CREDIT`。
- 用户问当日委托里“只看可撤单”，优先指向 `cancelable_only=True`。

# xtquant.xttrader 约券与回调

来源：[XtQuant.Xttrade 交易模块](https://dict.thinktrader.net/nativeApi/xttrader.html?id=olYCD5)

## 约券相关接口

### smt_query_quoter(account)

- 作用：券源行情查询。
- 返回：券源信息列表。
- 常见字段：
  - `success`
  - `error`
  - `finType`
  - `stockType`
  - `date`
  - `code`
  - `codeName`
  - `exchangeType`
  - `fsmpOccupedRate`
  - `fineRate`
  - `usedRate`
  - `enableSloAmountT0`
  - `enableSloAmountT3`
  - `srcGroupId`
  - `applyMode`

### smt_negotiate_order_async(account, src_group_id, order_code, date, amount, apply_rate, dict_param={})

- 作用：库存券约券申请异步接口。
- 返回：请求序号 `seq`，成功时大于 `0`，失败为 `-1`。
- 后续：正常会收到 `on_smt_appointment_async_response` 回报。
- 可选 `dict_param` 字段：
  - `subFareRate`
  - `fineRate`

### smt_query_compact(account)

- 作用：约券合约查询。
- 返回：约券合约信息列表。
- 常见字段：
  - `success`
  - `error`
  - `createDate`
  - `cashcompactId`
  - `oriCashcompactId`
  - `applyId`
  - `srcGroupId`
  - `comGroupId`
  - `finType`
  - `exchangeType`
  - `code`
  - `codeName`
  - `date`
  - `compacAmount`
  - `compacBalance`
  - `returnAmount`
  - `usedAmount`
  - `usedRate`
  - `compactStatus`
  - `applyMode`

## 回调类

### on_disconnected()

- 作用：连接断开时推送。

### on_account_status(data)

- 参数：`XtAccountStatus`
- 作用：账号状态变动推送。

### on_stock_order(data)

- 参数：`XtOrder`
- 作用：委托状态、成交数量等变化推送。

### on_stock_trade(data)

- 参数：`XtTrade`
- 作用：成交变动推送。

### on_order_error(data)

- 参数：`XtOrderError`
- 作用：下单失败推送。

### on_cancel_error(data)

- 参数：`XtCancelError`
- 作用：撤单失败推送。

### on_order_stock_async_response(data)

- 参数：`XtOrderResponse`
- 作用：异步下单回报推送。

### on_smt_appointment_async_response(data)

- 参数：`XtSmtAppointmentResponse`
- 作用：约券相关异步接口回报推送。

## 最小回调示例

```python
from xtquant.xttrader import XtQuantTraderCallback


class MyXtQuantTraderCallback(XtQuantTraderCallback):
    def on_disconnected(self):
        print('connection lost')

    def on_account_status(self, status):
        print(status.account_id, status.account_type, status.status)

    def on_stock_order(self, order):
        print(order.stock_code, order.order_status, order.order_sysid)

    def on_stock_trade(self, trade):
        print(trade.account_id, trade.stock_code, trade.order_id)

    def on_order_error(self, order_error):
        print(order_error.order_id, order_error.error_id, order_error.error_msg)

    def on_cancel_error(self, cancel_error):
        print(cancel_error.order_id, cancel_error.error_id, cancel_error.error_msg)

    def on_order_stock_async_response(self, response):
        print(response.account_id, response.order_id, response.seq)
```

## 使用建议

- 用户说“异步下单返回了 seq 但没看到订单号”，说明还要等 `on_order_stock_async_response`。
- 用户说“没有任何推送”，优先检查 callback 是否注册、账号是否订阅、进程是否仍在 `run_forever()`。
- 用户说“约券申请失败”，优先看是否有 `on_smt_appointment_async_response` 回报和返回消息。
